# Деплой BondPulse на Yandex Cloud

## Пошаговая инструкция

### 1. Регистрация в Yandex Cloud
- Перейди на https://cloud.yandex.ru
- Войди через Яндекс ID (нужен аккаунт на ya.ru или yandex.ru)
- Получи бесплатный бонус 3000₽ на 30 дней

### 2. Создай облачный кластер
1. Перейди в **Облако** → **Создать облачный кластер**
2. Название: `bondpulse-cloud`
3. Нажми **Создать**

### 3. Создай Compute Engine (VPS)
1. Перейди в **Compute Cloud** → **Создать виртуальную машину**
2. Настройки:
   - **Название**: `bondpulse-server`
   - **Зона**: `ru-central1-a` (Москва)
   - **Платформа**: Intel Cascade Lake
   - **vCPU**: 2
   - **RAM**: 2 ГБ
   - **Диск**: 20 ГБ (SSD)
3. Образ: **Ubuntu 22.04**
4. Сетевые настройки:
   - **Внешний IP**: Публичный
5. Доступ: Выбери **Создать пароль** или **SSH-ключ**
6. Нажми **Создать виртуальную машину**

### 4. Подключись по SSH
```bash
ssh root@ВНЕШНИЙ_IP_СЕРВЕРА
```

### 5. Запусти деплой (одна команда)
```bash
bash <(curl -s https://raw.githubusercontent.com/macromax2/bond-screener/main/deploy-yandex.sh)
```

### 6. Проверь
Открой в браузере: `http://ВНЕШНИЙ_IP_СЕРВЕРА`

---

## Бонусный кредит 3000₽

Яндекс даёт **3000₽** на 30 дней бесплатно. Этого хватит на:
- Compute Engine (2 vCPU, 2GB): ~600₽/мес
- Managed PostgreSQL: ~800₽/мес
- Итого: ~1400₽/мес → **останется кредит на 2 месяца**

### Как получить бонус:
1. После регистрации перейди в **Биллинг**
2. Нажми **Пополнить баланс**
3. Бонус начислится автоматически
